# Output Contract

## Summary
Defines the expected output structure for reuse decisions.

---

## Rule: Structured Decision Output
**Rule ID:** rru-output  
**Priority:** MUST  
**Applies to:** react_reuse_update_new  
**Rationale:** Ensures decisions are actionable and traceable.

### Requirement

- Output must state reuse/update/new for each relevant artifact.
- Output must include short reasons for each decision.
- Output must list any risks or constraints.

### Forbidden

- Output that hides the decision rationale.
- Ambiguous decisions without clear action.

### Notes

- Keep notes concise and focused on material tradeoffs.
