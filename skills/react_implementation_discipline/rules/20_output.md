# Output Contract

## Summary
Defines the expected output structure for implementation discipline results.

---

## Rule: Structured Implementation Output
**Rule ID:** rid-output  
**Priority:** MUST  
**Applies to:** react_implementation_discipline  
**Rationale:** Ensures results are verifiable and traceable.

### Requirement

- Output must list files changed and checks performed.
- Output must report boundary audit outcomes.
- Output must include any deviations or unresolved risks.

### Forbidden

- Omitting validation results.
- Returning unstructured prose instead of structured output.

### Notes

- Keep outputs concise and focused on verification.
