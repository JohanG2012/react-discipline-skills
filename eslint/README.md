# ESLint Recommendations

This directory provides recommended ESLint rule sets for supported versions.
Merge the appropriate configuration into your existing ESLint setup.

- **ESLint 8.50.0**: `eslint/8.50.0/.eslintrc.cjs`
- **ESLint 10.0.0**: `eslint/10.0.0/eslint.config.cjs`

These recommendations focus on architecture boundaries, import discipline, and
safe defaults for skill usage.
