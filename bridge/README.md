# Bridge Integrations

This folder contains integration bridges for external planning workflows.

- `speckit-rules-bridge/`: local Spec Kit extension that adds a rule-aware
  clarify command (`speckit.rules-bridge.clarify`).
- `install-speckit-clarify-bridge.sh`: installer script to register the
  extension in a target Spec Kit project.
- `uninstall-speckit-clarify-bridge.sh`: uninstall script to disable/remove the
  extension from a target Spec Kit project.
